# Project Parallel v1.1.9

Correcao do fluxo de autoatualizacao do desktop para o beta aberto.

## Destaques

- updater ajustado para preferir o `Setup` silencioso nas proximas atualizacoes
- corrigido o restart automatico apos a atualizacao
- criado `update.log` na pasta da instalacao para facilitar diagnosticos
- mantido o `Portable .zip` como fallback quando a release nao tiver instalador

## Artefatos desta release

- `ProjectParallel_Setup_v1.1.9.exe`
- `ProjectParallel_Portable_v1.1.9.zip`

## Observacoes

- para quem esta na `v1.1.8`, a recomendacao e instalar esta versao pelo `Setup` uma vez
- a partir da `v1.1.9`, o auto-update passa a usar o `Setup` silencioso quando ele estiver disponivel na release
- o arquivo `Portable .zip` segue como fallback tecnico para releases sem instalador
- esta release substitui a `v1.1.8` como versao recomendada para o beta
