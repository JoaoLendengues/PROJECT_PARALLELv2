# Project Parallel v1.1.8

Beta aberto com ajustes nos widgets, no instalador e no fluxo de atualizacao do desktop.

## Destaques

- novo fluxo de build do desktop com PyInstaller
- novo instalador Windows com Inno Setup
- pacote portatil preparado para distribuicao e atualizacao automatica
- updater ajustado para atualizar o build empacotado do Windows
- leitura de recursos e configuracoes adaptada para o executavel instalado
- correcoes no carregamento da URL da API e no status do servidor no desktop

## Artefatos desta release

- `ProjectParallel_Setup_v1.1.8.exe`
- `ProjectParallel_Portable_v1.1.8.zip`

## Observacoes

- o `Setup` e indicado para instalacao inicial do beta
- o arquivo `Portable .zip` e o pacote usado pelo sistema de atualizacao automatica
